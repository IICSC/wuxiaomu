## 本地运行

### 方法一：直接打开
双击 `index.html` 在浏览器中打开

### 方法二：使用VS Code Live Server
```bash
# 安装Live Server扩展
# 右键index.html -> Open with Live Server
```

### 方法三：使用Python
```bash

cd wuxiaomu-profile

# 启动本地服务器
python -m http.server 8080

# 访问 http://localhost:8080
```
